Welcome to the
SGS Scavenger Hunt!

This is a physical web beacon-enabled hunt around the SGS Office.

How it works

When the game is started, a timer will begin, and a clue will appear.

Search for the objects

Try to determine the object in the office that the clue is describing. When you have found the correct object, your phone will be sent a message from the object's beacon.

Can't find the object?

If you can't find the object from the first clue, you will be able to request 2 more clues. However, the more clues you request, the less points you get!

Winning the Game

The goal is to find all 4 objects using the least amount of clues, in the shortest time possible.

Submit your Score

When you have found all the objects, submit your score. The scores will be ranked and the winner will be announced after all the participants have finished!

Are you ready to begin?

Clicking the start button will begin the timer, so wait until you are ready to start the game before pressing it!
